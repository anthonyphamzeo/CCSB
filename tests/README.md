# Tests

| Suite | Scope |
|---|---|
| `Unit/` | Pure domain, application, adapter, plug-in, and UI unit tests |
| `Contract/` | Custom API, event, schema, and compatibility contracts |
| `Integration/` | Dataverse and Azure adapter behavior in controlled environments |
| `Architecture/` | Dependency direction, naming, forbidden references, and solution boundaries |
| `UI/` | PCF component, host, browser, accessibility, and keyboard journeys |
| `Performance/` | Plug-in budgets, visible-slice queries, payload size, and PCF rendering |
| `Fixtures/` | Deterministic synthetic datasets shared across suites |

Release evidence is published under `docs/implementation/testing/evidence/`; transient test output remains ignored.
