# Design Artefacts

| Folder | Purpose |
|---|---|
| `architecture/` | Context, container, component, sequence, environment, and solution-layering designs |
| `architecture/decisions/` | Numbered architecture decision records |
| `data-model/` | Dataverse tables, columns, keys, relationships, ownership, retention, and migration designs |
| `api-contracts/` | Versioned Custom API requests, responses, errors, idempotency, and concurrency contracts |
| `integrations/` | Service Bus, Functions, webhooks, external systems, and failure/retry designs |
| `security/` | Roles, privileges, field security, identities, threat models, and audit requirements |
| `ux/` | Model-driven app, PCF interaction, accessibility, and responsive-state designs |
