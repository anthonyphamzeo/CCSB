# Release 01: Native Schedule Board MVP

## Goal

Prove one native CCSB board with governed server operations, the Runtime PCF, multi-resource assignment, targeted refresh, and minimum audit evidence.

## Required foundations

- Source-controlled managed-solution baseline and dependency manifest.
- Physical `ccsb_*` schema and compatibility guardrails.
- Security baseline and versioned Custom API envelope.
- Configuration Studio boundary and deployable shell only.
- Runtime PCF and fail-closed projection controller.
- Controlled schedule operations and release evidence.

Detailed task selection is populated from the approved release backlog; no task is considered complete solely because it appears in this list.
