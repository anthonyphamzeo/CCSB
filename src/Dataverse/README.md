# Dataverse

- `Solutions/CCSB.Core`: publisher, common schema/configuration, security primitives, and shared server components.
- `Solutions/CCSB.Scheduling`: scheduling schema, Custom APIs, plug-ins, audit, publish, and rollback.
- `Solutions/CCSB.ModelDrivenUX`: model-driven app, forms/views, commands, web resources, and PCF registrations.
- `EnvironmentVariables`: definitions and non-secret defaults.
- `Security`: role and field-security source/evidence.
- `Data/Reference`: deployable reference/configuration data.
- `Data/Migration`: repeatable transforms and migration maps.
- `Data/TestData`: deterministic non-production fixtures.

Solution directories contain unpacked source only. Managed/unmanaged package ZIPs are build artefacts.
