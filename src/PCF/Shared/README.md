# PCF Shared Code

This folder is for UI-only TypeScript shared across Planora PCF controls.

Allowed:

- Custom API client wrappers
- Fluent UI wrappers and theme tokens
- Accessibility helpers
- Telemetry adapters
- Test utilities

Not allowed:

- Server-side scheduling rules
- Dataverse transaction logic
- Security decisions
- Compatibility validation rules that belong in Dataverse services
